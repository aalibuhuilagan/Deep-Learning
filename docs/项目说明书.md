# 项目说明书

## 一、项目基本信息

### 项目名称

揭秘 AI 的视觉焦点 —— 基于 Grad-CAM 的图像可解释性可视化系统

### 项目类型

深度学习课程设计

### 开发环境

- 操作系统：Windows 10 / Windows 11
- 开发工具：PyCharm
- Python 版本：3.10
- 运行环境：Anaconda 虚拟环境

### 技术架构

- 界面层：PyQt5
- 算法层：PyTorch + Grad-CAM / Grad-CAM++
- 图像处理：OpenCV、Pillow

## 二、项目背景

深度卷积网络、视觉 Transformer 在图像分类任务中精度很高，但属于黑盒模型。

使用者无法直观了解模型依据图像哪些区域做出判断。

Grad-CAM 是主流可解释性算法，通过热力图高亮模型视觉关注点。

本项目基于该算法搭建可视化 GUI 工具，完成课程设计实践。

## 三、整体功能设计

### 1. 基础模块

实现模型加载、单张图片导入、热力图生成、结果展示与自动保存。

### 2. 进阶模块

1. 批量图片处理：批量生成多张图片热力图
2. 视频分析：读取本地视频，逐帧叠加视觉热力图
3. 摄像头分析：调用摄像头实时可视化
4. 多算法对比：同屏展示原图、Grad-CAM、Grad-CAM++ 效果

### 3. 支持模型

- ResNet50
- VGG16
- ViT-B/16

## 四、运行流程

1. 启动程序，进入主界面
2. 选择【基础功能】或【进阶功能】选项卡
3. 进阶功能先选择并加载模型
4. 选择图片 / 视频或启用摄像头，执行对应功能
5. 查看结果，文件自动保存至输出目录

## 五、项目特点

1. 图形化界面，操作简单易用
2. 兼容卷积网络与视觉 Transformer 两类模型
3. 多场景支持：单图、批量、视频、摄像头
4. 多算法横向对比，直观展示效果差异
5. 代码模块化，结构清晰，便于扩展
