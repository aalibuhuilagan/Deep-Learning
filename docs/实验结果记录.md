# 实验结果记录

## 一、实验基本信息

### 实验名称

基于 Grad-CAM 系列算法的模型注意力可视化对比实验

### 实验环境

- 框架：PyTorch 2.0.1
- 测试模型：ResNet50、VGG16、ViT-B/16
- 测试算法：Grad-CAM、Grad-CAM++
- 输入图像尺寸：224 × 224

### 实验目的

1. 验证算法对不同模型的适配能力
2. 对比两种算法的可视化效果差异
3. 观察卷积网络与 Transformer 注意力分布区别

## 二、实验一：ResNet50 模型

### Grad-CAM 效果

热力图可以完整覆盖目标主体，定位基本准确；

但关注区域范围偏大，会附带少量背景区域。

### Grad-CAM++ 效果

热力轮廓更贴合物体边缘，区域更加集中；

几乎不会引入无关背景，定位精度更高。

### 实验结论

ResNet50 特征提取能力优秀，两种算法均可正常解释模型决策；

Grad-CAM++ 综合表现优于标准 Grad-CAM。

## 三、实验二：VGG16 模型

VGG16 注意力区域整体偏集中。

Grad-CAM 依旧存在区域偏大的问题。

Grad-CAM++ 对局部特征、小目标捕捉效果更好。

结论：VGG16 与 ResNet 表现趋势一致。

## 四、实验三：ViT-B/16 模型

ViT 注意力分布和传统卷积网络有明显区别。

本项目代码完成特征重塑适配，算法可以正常生成有效热力图。

两种算法均能定位图像关键区域。

结论：CAM 算法可正常用于视觉 Transformer 模型。

## 五、算法对比总结

|        算法         |                     优点                   |                   缺点              |                适用场景             |

| Grad-CAM     | 速度快、开销小、通用性强 | 区域偏大、易包含背景 | 快速可视化、大批量处理 |

| Grad-CAM++ |        定位精细、边缘准确     |           计算开销略高       | 细粒度分析、多目标场景 |

## 六、整体实验结论

1. Grad-CAM 与 Grad-CAM++ 均可稳定运行，适配 CNN 与 ViT 模型。
2. 热力图可有效高亮模型视觉区域，证明模型依据图像主体完成分类，决策逻辑合理。
3. Grad-CAM++ 是 Grad-CAM 的有效改进，定位精度提升明显。
4. 整套可视化工具功能稳定，所有模块均达到设计要求。
