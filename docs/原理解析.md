1. # Grad-CAM 及相关算法原理解析

   ## 一、研究背景

   深度神经网络在图像分类领域表现优异，但存在**可解释性差**的问题。

   外界无法得知模型依据图像哪一部分特征完成分类，这类模型被称为黑盒模型。

   Class Activation Mapping（类激活映射）系列算法，通过热力图可视化模型的视觉关注区域，

   是目前深度学习可解释性最常用的实现方案。

   本项目主要实现 Grad-CAM 与 Grad-CAM++ 两种经典算法，并完成对比实验。

   ## 二、前置基础知识

   1. 卷积特征图

      卷积层会逐步提取图像边缘、纹理、形状、高层语义特征。

      网络最后几层特征图，对应模型用于分类的核心语义信息。

   2. 梯度含义

      反向传播过程中，分类结果对特征图的梯度，代表该区域对分类决策的贡献大小。

      梯度数值越大，对应区域越被模型重视。

   3. 热力图规则

      项目使用 JET 伪彩色图谱：

   - 红、黄色区域：模型重点关注区域
   - 蓝、冷色区域：模型关注度较低

   ## 三、Grad-CAM 算法原理

   ### 3.1 算法整体流程

   Grad-CAM 全称为梯度加权类激活映射，无需修改网络结构，通用性极强。

   完整步骤：

   1. 前向传播

      将预处理图片输入模型，完成推理，保留网络最后一层特征图。

   2. 反向传播

      针对预测类别，计算分类得分对特征图的梯度。

   3. 计算通道权重

      对每个通道的梯度做全局平均池化，得到各通道的重要权重。

   4. 特征加权融合

      使用权重对所有特征通道加权求和，得到单通道激活图。

   5. 过滤无效区域

      使用 ReLU 函数剔除负梯度区域，负梯度代表抑制分类，无可视化意义。

   6. 缩放与叠加

      将激活图缩放到原图尺寸，转为伪彩色热力图，半透明叠加在原图上。

   ### 3.2 算法优缺点

   优点：

   - 无需修改网络结构，适配绝大多数图像模型
   - 计算速度快，资源消耗低
   - 能够准确定位目标主体

   缺点：

   - 关注区域范围偏大
   - 容易附带部分背景干扰
   - 对细小目标、多目标定位精度一般

   ## 四、Grad-CAM++ 算法原理

   ### 4.1 优化方向

   针对 Grad-CAM 定位粗糙、边缘不清晰的问题，Grad-CAM++ 优化梯度计算逻辑。

   ### 4.2 核心改进

   1. 引入二阶、三阶梯度信息，丰富特征权重计算依据
   2. 对梯度做归一化处理，降低背景干扰
   3. 优化权重分配，让热力图更聚焦目标本体

   ### 4.3 效果对比

   - Grad-CAM：区域偏大，边缘模糊
   - Grad-CAM++：轮廓精细，边缘清晰，小目标识别效果更好

   ## 五、项目内部实现流程

   1. 图片读取与预处理：缩放、归一化、转为模型张量
   2. 模型前向推理，捕获特征图与梯度
   3. 根据选择算法生成单通道激活图
   4. 转换为伪彩色热力图
   5. 半透明叠加在原始图片上
   6. 界面展示 + 文件保存

   ### 关键适配点

   1. 模型状态切换

      计算梯度时临时开启训练模式，计算完成恢复推理模式，保证模型正常使用。

   2. ViT 模型适配

      视觉 Transformer 无传统卷积特征图，通过特征重塑函数适配 CAM 计算逻辑。

   3. 图像叠加

      使用 OpenCV 加权融合，固定透明度，保证原图色彩不被覆盖。

   ## 六、应用场景

   1. 深度学习模型可解释性分析
   2. 模型效果排查，判断模型是否关注无效背景
   3. 医学影像、自动驾驶等高安全要求场景验证
   4. 教学演示，直观展示网络注意力机制

   ## 七、总结

   Grad-CAM 系列是深度学习可解释性的经典算法。

   本项目通过可视化工具，直观对比两种算法差异，同时拓展多种实用功能，

   完整实现课程设计的学习与实践目标。
